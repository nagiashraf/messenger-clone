# Messenger Clone Application

## Features

- Conversations

  - Create one-to-one and group conversations
  - Delete conversations

- Messages

  - Send and receive real-time messages

- Authentication

  - Registeration and Login
  - Google Authentication

## Tech Stack

- Laravel
- MySQL
- Inertia.js
- React.js
- Tailwind CSS
- Headless UI
- React-Hook-Form
